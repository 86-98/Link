# **App Name**: AI Oracle

## Core Features:

- Expert Listing: Display a clean, visually appealing list of AI experts.
- Profile View: Each expert profile shows a brief bio and their predictions.
- Navigation: Easy navigation to browse different experts and predictions.
- Recommendation Engine: AI tool suggests related experts or similar predictions based on user interaction.
- Filtering/Sorting: Ability to filter and sort experts based on expertise.

## Style Guidelines:

- Primary color: Yellow, orange , red, white
- Background color: Light Gray (#F0F0F5), providing a clean, distraction-free background.
- Accent color: Teal (#009688), offering a modern, tech-oriented feel to highlight interactive elements.
- Body font: 'Inter' (sans-serif) for a clean and modern reading experience. Headline Font: 'Space Grotesk' (sans-serif) for a technological aesthetic.
- Simple, professional icons to represent different fields of AI and machine learning.
- Clean, card-based layout for expert profiles with clear sections for bio and predictions.
- Subtle transitions and animations for smooth navigation and engagement.
